# Official Source Instances Selection

## Primary official source profile

- Kyiv emergency / air alert service (nooddienst)
- Homepage or practical feed acceptable (need not be perfect RSS)
- Semi-manual processing is acceptable for day 1

## Fallback official source profile

- National Ukrainian emergency / air alert service
- Used only if primary source is unavailable or unusable

## UI labeling

- Unified label for both sources in demo: `Kyiv Official Alerts`

## Next step process

- Agent will propose candidate instances that match this profile
- User approves each candidate before it is used in implementation
- Candidate sources must include at least their public name and homepage/feed URL
